## 💜 About the Project

This project was developed as part of the **Santander - Excel with Artificial Intelligence Bootcamp** on the DIO platform.

The objective was to create a **practical investment simulation tool** using Excel, focused on real estate funds (FIIs), enabling users to make **data-driven financial decisions**.

This solution was designed under the concept of **NYKOS CAPITAL**, a fictional brand created to represent sophistication, strategy, and long-term financial vision.

## ▶️ How to Use

1. Download the Excel file
2. Open in Microsoft Excel
3. Input your investment parameters:

   * Initial amount
   * Monthly contribution
   * Time period
   * Expected return rate
4. Analyze the generated results and projections
